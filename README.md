## 🛠️ Installation & Setup  

### 1️⃣ Clone the repository (or extract ZIP from Drive)  
```bash
git clone <your-repo-or-folder>
cd project-folder
```

### 2️⃣ Backend Setup  
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 3️⃣ Frontend Setup  
```bash
cd frontend
npm install
npm start
```

The app will run at:  
- Frontend → `http://localhost:5173`  
- Backend → `http://localhost:4000`
